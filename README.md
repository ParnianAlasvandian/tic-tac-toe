# ⭕❌ Tic Tac Toe (Python)

## About

A console-based Tic Tac Toe game built in Python focusing on game logic, user interaction, and structured programming.

---

## 🎮 Features

* Two-player gameplay (Player 1 vs Player 2)
* Input validation for moves
* Turn-based system
* Win detection (rows, columns, diagonals)
* Draw detection (tie game)
* Replay functionality
* Clean console-based UI

---

## 🛠️ Technologies Used

* Python 3
* Basic control flow (loops & conditionals)
* Functions
* Random module (for first player selection)

---

## 📂 Project Structure

```
Tic Tac Toe Project
│
├── display_board()      → Shows game board
├── player_input()       → Marker selection (X or O)
├── place_marker()       → Updates board
├── win_check()          → Checks winning conditions
├── helper functions     → Game logic utilities
└── main game loop       → Controls full gameplay
```

---

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/ParnianAlasvandian/tic-tac-toe.git
```

2. Navigate to project folder:

```bash
cd tic-tac-toe-python
```

3. Run the game:

```bash
python tic_tac_toe.py
```
